# Семинар 01 – Dev-среда, Git, шаблонный репозиторий, NumPy

## 1. Место в курсе

Первый семинар курса «Инженерия Искусственного Интеллекта».

Опирается на материалы Лекции L01 (обзор курса, роли, жизненный цикл ИИ-систем) и подготавливает базу для всех последующих недель:

- рабочая среда для курса;
- единая структура студенческого репозитория;
- базовые навыки работы с Git;
- минимально необходимый уровень владения NumPy.

Домашнее задание по семинару — **HW01**, его формальное описание дано в файле `seminars/S01/S01-homework.md`.  
Выполнение HW01 — технический «входной билет» к остальным семинарам и сквозному проекту.

---

## 2. Цели семинара

После семинара и выполнения домашнего задания студент:

- умеет настроить рабочую среду под курс (Python, виртуальное окружение, IDE/Jupyter, базовые библиотеки);
- понимает принцип работы Git и может выполнить базовый цикл: `clone → status → add → commit → push`;
- понимает структуру и назначение шаблонного репозитория `aie-student-template`;
- умеет создавать и изменять массивы NumPy и выполнять над ними базовые операции.

---

## 3. Краткое содержание

1. **Вступление**
   - связь с лекцией L01 и целями курса;
   - как будут использоваться репозитории и домашние задания в течение семестра.

2. **Настройка окружения**
   - проверка версии Python;
   - создание и активация виртуального окружения;
   - базовая установка необходимых пакетов;
   - запуск IDE / Jupyter (минимальный рабочий сценарий).

3. **Введение в Git и GitHub**
   - что такое репозиторий;
   - локальный и удалённый репозиторий;
   - типичный цикл работы: `clone`, `status`, `add`, `commit`, `push`;
   - короткий разбор типичных ошибок и как их не усугублять.

4. **Обзор шаблонного репозитория `aie-student-template`**
   - структура каталогов;
   - папка `homeworks/` для домашних заданий `HWNN`;
   - папка `project/` для сквозного мини-проекта;
   - ключевые файлы и как ими пользоваться вместе с гайдами из `guides/`.

5. **Базовые операции в NumPy**
   - создание массивов из списков и с помощью конструкторов (`zeros`, `ones`, `arange` и т.п.);
   - форма (`shape`) и типы (`dtype`);
   - индексирование и срезы;
   - простые векторные операции и базовые статистики (`mean` и аналоги).

---

## 4. Формат проведения

Семинар проводится в формате, согласованном с общей политикой по семинарам:

- **Краткий разбор**:
  - напоминание ключевых идей лекции L01;
  - формулировка задач занятия и связи с HW01.

- **Практическая часть (live-coding)**:
  - демонстрация команд в терминале и Git;
  - показ работы с шаблонным репозиторием;
  - базовые примеры NumPy в ноутбуке / скрипте.

- **Постановка домашнего задания HW01**:
  - пояснение структуры `homeworks/HW01/` в студенческом репозитории;
  - акцент на требования к именованию и публичности репозитория;
  - ответы на технические вопросы (Git, среда, NumPy).

Студенты могут частично повторять шаги за преподавателем, но финальное оформление и закрепление материала предполагается сделать **самостоятельно дома** при выполнении HW01.

---

## 5. Результат для студента

К концу семинара студент:

- понимает, как он будет работать с материалами курса через Git и GitHub;
- знает, где в его репозитории будут лежать домашние задания (`homeworks/HWNN/`) и сквозной проект (`project/`);
- видел «эталонный» пример базовой работы с NumPy, который может воспроизвести дома.

После выполнения домашнего задания HW01 у студента:

- создан публичный репозиторий на GitHub на основе `aie-student-template`;
- настроен локальный доступ к репозиторию через Git;
- в репозитории есть папка `homeworks/HW01/` с первым ноутбуком/скриптом по NumPy и тестовым коммитом.
